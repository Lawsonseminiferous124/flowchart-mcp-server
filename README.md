## What this software does

The flowchart-mcp-server helps you turn text or code into visual flowcharts. You use it with the Claude AI assistant to map out complex ideas, organize workflows, or visualize processes. It creates academic-quality diagrams that remain easy to read. This tool connects your thoughts to a visual format. It works locally on your machine for better speed and privacy.

## 💻 System requirements

Before you install this software, confirm your computer meets these needs:

*   **Operating System:** Windows 10 or Windows 11.
*   **Memory:** At least 4 gigabytes of RAM.
*   **Internet Access:** Required for the initial setup and to connect with the Claude AI model.
*   **Storage:** 200 megabytes of free space on your hard drive.

## 🎨 Creating a flowchart

After the server runs, you connect it to your Claude interface.

1.  Open the Claude application or website.
2.  Access your settings or preferences menu.
3.  Look for the "MCP" or "Model Context Protocol" section.
4.  Add a new server connection by pointing it to the local folder where you installed this software.
5.  Once connected, you can type prompts like "Create a flowchart for the sign-up process" directly into the Claude chat window.
6.  The server generates the diagram as an image file that you can save or copy into your documents.

## 🔍 Troubleshooting common issues

If you encounter problems, check these items first:

*   **The window closes immediately:** Ensure that you have the latest version of Python installed on your Windows machine. You can get this from the official Python website if the setup fails to start.
*   **Claude does not see the server:** Confirm the black window is still open. If you closed it, the connection drops. Restart the program to restore the link.
*   **The diagram looks incomplete:** Check your input. Using clear, step-by-step instructions helps the AI build a better chart. Avoid vague descriptions to get the best results.
*   **Permission errors:** If Windows blocks the file, right-click the icon and choose "Run as administrator." This gives the program permission to create the files necessary for saving your flowcharts.

## 📝 Tips for better charts

*   **Be specific:** Instead of saying "make a graph," say "make a flowchart of the steps to bake a cake."
*   **Use code:** If you have data in a format like CSV or JSON, paste that into the chat. The server reads this data directly and builds accurate diagrams based on your specific numbers.
*   **Update often:** Check the releases page every few weeks. We update the server to include new shapes and better layout options for your diagrams.
*   **Check your connection:** Because this requires an AI model, ensure your internet connection stays stable while the server processes your request. If your internet cuts out mid-process, the server may need a restart.

## 🛡️ Privacy and your data

This server processes information locally on your machine. The diagram generation happens through the Model Context Protocol, which sends your prompt to Claude. It does not store your charts on a public server. You remain the owner of the images you create. You can delete the files from your computer at any time without affecting your access to the software.

## 🛠️ Advanced configuration

Most users run the program with default settings. If you need to change how the diagrams look, locate the configuration file in your installation folder. This file uses simple text to define things like colors, fonts, and chart orientation. Open this file in Notepad, change the values, and restart the server to see your new styles. Do not modify other files in this folder, as they are required for the program to run correctly. 

If you make a mistake in the configuration file, delete it and restart the application. The software automatically replaces it with the original version.
